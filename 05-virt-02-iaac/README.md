Задача 1
=====

* **Опишите своими словами основные преимущества применения на практике IaaC паттернов.**

**Ответ.**
```
Основные приемущества - снижение цены устранения ошибки в разработке путем непрерывного автоматизированного 
слияния веток разработки и тестирования внесенных изменений.
Уменьшение рутинных задач за счет автоматизации процессов по слиянию веток и проведения тестов
Уменьшение финансовых и времязатрат на утверждение кода (в ветках для разработчиков). 
т.е. основная задача - максимально автоматизировать процесс разработки, где это возможно без последствий. оставить подтверждение изменений
там, где существует риск для бизнеса от возникших в результате разработки ошибок в коде
```
* **Какой из принципов IaaC является основополагающим.**

**Ответ.**
```
На мой взгляд основополагающим принципом IaaC является устранение дрейфа конфигураций. т.е. главное - настроить инфраструктуру
таким образом, чтобы работа системы была предсказуемой, изменния конфигураций носили прозрачный характер, чтобы в инфраструктуре не было места несовпадению сред 
разработки, тестирования и развертывания. При соблюдении этого принципа разработка будет эффективной, программный продукт будет
быстрее производиться и доставляться до потребителя.
```
